# CursoJavaSence-Fullstack
Repositorio con los proyectos del curso Java Fullstack de Sence desarrollados por mí.
# Como usar el repo
Al descargar el repositorio y hostearlo directamente con Live Server en VS code (originalmente usé la extensión Five Server), debe entrar en la carpeta correspondiente al proyecto del módulo. Por el momento solo está el proyecto del módulo 2, por lo que al hostear el repositorio completo debe hacer click en la carpeta "M2" en el navegador para ejecutar la página index.html automáticamente, que es login de la wallet. Para el caso en que hayan más proyectos deberá hacer click a la carpeta asociada al módulo (Ej: "M3"= proyecto del módulo 3, etc).

# Proyecto del módulo 2
Las credenciales para logearse son "123" como usuario y contraseña. Si intenta ingresar otro le avisara que las credenciales son incorrectas. De ahí en adelante puede probar todas las funcionalidades disponibles.
## Menú
En el menu verá su saldo y las otras página para hacer depósitos a la cuenta, transferir a otros contactos o ver los últimos movimientos.
## Depósitos
Ingrese una cantidad y deposite para aumentar su saldo.
## Enviar dinero
Es para enviar dinero a sus contactos. Puede agregar nuevos o seleccionar los que ya tiene agregados para enviarles dinero
## Últimos movimientos
Mustra todos los movimientos en la cuenta y se pueden filtrar según el tipo. El tipo compras siempre estará vacío ya que no está esa funcionalidad pero fue implementó el filtro para cuando exista la funcionalidad de compras.
## Notas
El saldo, contactos y movimientos están guardados en el localStorage, por lo que debiera de borrarlos cuando ya deje de utilizar la página.
Se usó chrome al programarlo, por lo que se recomienda usarlo también.
